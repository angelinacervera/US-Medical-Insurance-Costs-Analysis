# U.S. Medical Insurance Costs Analysis

## Project Overview
This project performs a deep-dive analysis into a dataset of medical insurance costs to identify key drivers of higher premiums. Using Python's `csv` library and object-oriented programming (OOP), I investigated the correlation between lifestyle choices, demographics, and healthcare expenses.

## Key Findings
* **The Smoking Premium:** Smokers pay significantly more across all age groups, with the "premium" increasing as patients get older.
* **The BMI Tipping Point:** For smokers, medical charges spike by over $20,000 once BMI crosses the 30.0 threshold.
* **Regional Variance:** The Southeast region shows the highest average insurance costs compared to the Northwest and Southwest.

## Skills Demonstrated
* **Data Cleaning & Loading:** Efficiently loading CSV data into structured Python dictionaries.
* **Object-Oriented Programming (OOP):** Building a reusable `PatientAnalysis` class.
* **Descriptive Statistics:** Calculating averages and variances to extract business insights.
